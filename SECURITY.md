# Security Policy

## Supported Versions

Current focus: `v0.1.x`

## Reporting a Vulnerability

1. Do not disclose vulnerabilities publicly before a fix is available.
2. Open a private report through GitHub Security Advisories if available.
3. If private reporting is unavailable, open an issue with minimal sensitive detail and request private follow-up.

## Security Notes for This Extension

1. Scope is limited to `https://chatgpt.com/*`.
2. Exported conversation content stays local unless a user manually shares files.
3. No remote analytics or telemetry endpoint is implemented in `v0.1.0`.
