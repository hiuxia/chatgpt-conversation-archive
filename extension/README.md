# Extension Quick Start

## Load Unpacked

1. Open `chrome://extensions`
2. Enable `Developer mode`
3. Click `Load unpacked`
4. Select this `extension/` folder

## Current Capabilities

1. Side panel button: export current conversation to Markdown
2. Side panel button: read currently loaded history links from sidebar DOM
3. Select history items with search/filter and export selected conversations to one ZIP

## Notes

- This is an MVP scaffold.
- Batch export currently opens selected conversation pages sequentially in background tabs.
